StreamlineVPN

Fetch, deduplicate, enhance, and export VPN configs in multiple formats with a single command or via a FastAPI backend.

Get Started View API Specs Run in Browser

Run via CLI

Quickly process VPN configurations from the command line.

pip install -r requirements.txt
python -m streamline_vpn

Start the API

Run the FastAPI server for a more interactive experience.

uvicorn streamline_vpn.web.api:create_app --factory

Key Features

  • ✅ Fetch from multiple sources
  • ✅ Deduplicate and merge
  • ✅ Geolocation-based filtering
  • ✅ Multiple output formats
  • ✅ Extensible plugin system
