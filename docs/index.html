Modern aggregation of VPN configurations

Fetch, deduplicate, enhance, and export VPN configs in multiple formats with a single command or via a FastAPI backend.

Run via CLI Start API OpenAPI
Tip: Start the API to see live health below.

Quick Start

Create a virtualenv and install dependencies
python -m venv .venv
source .venv/bin/activate   # Windows: .\.venv\Scripts\Activate.ps1
pip install -U pip
pip install -r requirements.txt
Run the merger and generate outputs
python -m streamline_vpn --config config/sources.yaml --output output --format json --format clash
Start the FastAPI server
uvicorn streamline_vpn.web.api:create_app --factory --host 0.0.0.0 --port 8000 --reload
